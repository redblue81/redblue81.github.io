Inserire codice HTML nei commenti di WordPress

Posted by RedBlue on March 23, 2012 · 1 min read

Oggi voglio parlare di un plugin per WordPress che consente di inserire codice HTML nei commenti: Preserve Code Formatting.

Il problema è semplicissimo: quando proviamo ad inserire del codice HTML in un commento, questo di default viene interpretato e quindi non è direttamente visibile (ad esempio se inseriamo il codice che disegna un rettangolo di colore rosso, nel commento vedremo il rettangolo di colore rosso e non il relativo codice). Per ovviare a questo comportamento ho scovato il plugin in questione, che presenta una configurazione banale, che personalmente ho deciso di non modificare rispetto al default:

In questo modo, nei commenti il codice non viene interpretato, ma visualizzato come avviene nel corpo degli articoli. Volendo immagino si possa anche customizzare un po' la visualizzazione, ma sinceramente non mi ci sono cimentato..
