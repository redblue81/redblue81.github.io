Udev-notify: aggiungere una notifica all'inserimento di una periferica USB in Ubuntu 11.04

Posted by RedBlue on May 14, 2011 · 1 min read

Oggi voglio proporvi una piccola utility che estende le funzionalità di Notify-OSD, il servizio di notifiche di Ubuntu. Si tratta di Udev-notify, che visualizza una notifica ogni volta che si connette o si disconnette una periferica USB al pc (telefoni cellulari, chiavette, fotocamere e così), come mostrato qui sotto..

Semplicemente basta aggiungere il relativo repository, installare il pacchetto e terminare la sessione; al nuovo login potrete sperimentare le nuove notifiche..

echo "deb http://download.learnfree.eu/repository/skss / #SKSS" | sudo tee -a /etc/apt/sources.list
wget http://download.learnfree.eu/repository/skss/repo.pub.asc -q -O- | sudo apt-key add -
sudo apt-get update
sudo apt-get install udev-notify

Alla prossima..
