Repository per Ubuntu 10.10 "Maverick Meerkat"

Posted by RedBlue on October 12, 2010 · 1 min read

Dopo il rilascio della nuova Ubuntu, sono anche stati ufficializzati molti nuovi repository. Qui ne voglio citare alcuni che ritengo importanti e di uso quotidiano (sono comprese anche le procedure di autenticazione):

Medibuntu (software multimediale non incluso in Ubuntu per motivi di licenze):

sudo wget --output-document=/etc/apt/sources.list.d/medibuntu.list http://www.medibuntu.org/sources.list.d/$(lsb_release -cs).list

wget http://packages.medibuntu.org/pool/free/m/medibuntu-keyring/medibuntu-keyring_2008.04.20_all.deb

sudo dpkg -i medibuntu-keyring_2008.04.20_all.deb

sudo apt-get update

Virtualbox (software per la creazione e l'utilizzo di macchine virtuali):

echo "deb http://download.virtualbox.org/virtualbox/debian maverick non-free #VirtualBox" | sudo tee -a /etc/apt/sources.list

wget -q http://download.virtualbox.org/virtualbox/debian/oracle_vbox.asc -O- | sudo apt-key add -

sudo apt-get update

sudo apt-get install virtualbox-3.2

GetDeb (molto software utile compilato e pacchettizzato):

wget -q -O- http://archive.getdeb.net/getdeb-archive.key | sudo apt-key add -

echo "deb http://archive.getdeb.net/ubuntu maverick-getdeb apps #GetDeb" | sudo tee -a /etc/apt/sources.list

sudo apt-get update

Bisigi Themes (temi per Gnome):

sudo add-apt-repository ppa:bisigi/dev

sudo apt-get update

sudo apt-get install bisigi-themes

Hotot (client Twitter che supporta OAuth):

sudo add-apt-repository ppa:hotot-team

sudo apt-get update

sudo apt-get install hotot

Ubuntu Tweak (potente utility di personalizzazione):

sudo add-apt-repository ppa:ubuntu-tweak-testing/ppa

sudo apt-get update

sudo apt-get install ubuntu-tweak
