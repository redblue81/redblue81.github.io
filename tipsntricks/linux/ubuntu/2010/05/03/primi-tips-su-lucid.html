Primi tips su Lucid..

Posted by RedBlue on May 03, 2010 · 1 min read

Per abilitare le icone nel menu Sistema, fino a Karmic c'era un comodo tasto da cui selezionale le relative opzioni. In Lucid è stato rimosso, ma è possibile ugualmente abilitare le icone con gconf-editor..

Da terminale, come utente normale, digitate

gconf-editor

Poi cercate la voce desktop/gnome/interface e selezionate le voci buttons_have_icons e menus_have_icons.

Ho riscontrato poi un piccolo bug in Evince, il visualizzatore di file PDF di Gnome. Semplicemente non apriva i file PDF, e lanciandolo da terminale restituiva l'errore:

** (evince:2420): WARNING **: Error creating last_settings file: Errore nell'aprire il file «/home/redblue/.gnome2/evince/last_settings»: Nessun file o directory

Segmentation fault

La soluzione è semplicemente quella di creare manualmente la directory evince..

mkdir /home/nomeutente/.gnome2/evince

Ovviamente sostituendo il vostro nome utente a nomeutente..

Alla prossima.. ;)
