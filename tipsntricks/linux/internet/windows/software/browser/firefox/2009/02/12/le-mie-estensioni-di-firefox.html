Le mie estensioni di Firefox

Posted by RedBlue on February 12, 2009 · 3 mins read

Mozila Firefox è il browser che detiene il record di downloads in un solo giorno, ormai più o meno tutti conoscono le sue potenzialità e i suoi punti di forza. Tra questi, c'è la possibilità di aggiungere al browser stardard una serie di estensioni, veri e propri programmi i quali implementano le funzioni più disparate, ce n'è per tutti, dall'utente che vuole semplicemente cambiare il tema, a chi vuole gestire il proprio blog, a chi vuole ascoltare musica, al webmaster professionista..

Ecco l'elenco delle estensioni che usa il sottoscritto:

  • AdBlock Plus: impedisce la visualizzazione di banner pubblicitari sulle pagine web visitate;
  • NoScript: impedisce l'esecuzione di script sulle pagine visitate (insieme ad AdBlock è uno strumento di sicurezza potentissimo);
  • ScribeFire: utilissimo blog editor. Vi permette di gestire un blog direttamente dal browser ed è compatibile con tantissime piattaforme (lo sto usando mentre scrivo questo post);
  • IE NetRenderer: vi permette di visualizzare vostre pagine web in Internet Explorer direttamente da Firefox (utilissimo per sviluppatori);
  • Foxmarks: avete una lista di preferiti chilometrica (come il sottoscritto)?? Se ad esempio formattate il pc dovete stare attenti a salvare il relativo file di Firefox, e se tale file va perso?? Ecco in aiuto Foxmarks, basta creare un account online e da quel momento ogni minima modifica ai preferiti sarà salvata online, in tal modo basterà semplicemente collegarsi al servizio e scaricare i propri preferiti; utilissimo anche a chi usa più di un pc;
  • FireBug: indispensabile per gli sviluppatori. Consente di effettuare il debug di una pagina web direttamente dal browser;
  • Download StatusBar: in realtà questa estensione non è proprio fondamentale, ma personalmente la trovo comoda.. Sostituisce la classica finestra di download con una discreta barra nella parte inferiore dello schermo;
  • Video DownloadHelper: utilissimo per scaricare i video dai siti stile Youtube;
  • DownThemAll!: praticamente un download manager per il browser, utilissimo per scaricare (anche "in blocco") tutto ciò che c'è in una pagina web.
  • User Agent Switcher: serve a cambiare l'user agent (avete presente quei siti che riconoscono il vostro browser in automatico? Ci riescono proprio attraverso questo parametro);
  • Forecastfox: le previsioni del tempo direttamente sul browser!!

In realtà questo elenco è fortemente incompleto, ci sono anche alcune altre estensioni che utilizzo (ad esempio la Google Toolbar), ma implementano funzioni estremamente specifiche, che non credo interessino alla maggior parte degli utenti, inoltre il "repertorio" è pressochè infinito, e mostruosamente adattabile ad gni esigenza, quindi non disperate se avete bisogno di una funzione particolare..
