In passato ho parlato della Gnome-shell Weather Extension (qui), ma ho trovato un fork di questo progetto che mi sembra decisamente interessante. E' mantenuto da Neroth ed è già provvisto di un comodissimo repository funzionante per Ubuntu. Purtroppo questa estensione non si trova sul sito dedicato extensions.gnome.org.

Inoltre, questa versione di Gnome-shell Weather Extension prevede un'interfaccia (che si trova in Strumenti di sistema -> Impostazioni meteo) per la configurazione della località e dell'aspetto.

Ancora, non c'è più bisogno di andare a cercare alcun WOEID o altro codice relativo alla località (viene tutto gestito in automatico); e infine, si può impostare più di una sola località di cui scaricare le informazioni relative al meteo.

Per chi usa Ubuntu, ecco le tre semplicissime istruzioni da seguire:

sudo add-apt-repository ppa:gnome-shell-extensions

sudo apt-get update

sudo apt-get install gnome-shell-extension-weather

Se invece, come me, usate Debian, allora potete scaricare i sorgenti da Git (qui), compilare ed installare il tutto col "solito" metodo:

git clone http://github.com/Neroth/gnome-shell-extension-weather

cd gnome-shell-extension-weather

./autogen.sh --prefix=/usr

make

sudo make install

In entrambi i casi, prima di procedere dovete riavviare Gnome-shell con la combinazione ALT+F2 e digitare la lettera r, poi attivate l'estensione tramite gnome-tweak-tool.

Per i più pigri, ho compilato e pacchettizzato il tutto, pronto per essere installato (su architettura a 64 bit). Lo trovate come al solito nell'area download e per installarlo basta dare, come root:

dpkg -i gnome-shell-extension-weather_3.4.1-1_amd64.deb
