Ubuntu Tweak è un pratico software che consente l'accesso a tutte quelle funzioni (desktop, repositories, programmi e via dicendo) per cui non esiste una specifica voce di menù, e che andrebbero perciò modificate con gconf.. Nella sua ultima versione, la 0.5.6 beta, sono state introdotte alcune caratteristiche decisamente interessanti, come la possibilità di resettare tutte le modifiche, riportando il sistema al momento dell'installazione del programma, e soprattutto la possibilità di effettuare il backup del desktop (ovvero solo di quelle opzioni che andremmo altrimenti a modificare con gconf), utilissimo ad esempio dopo una reinstallazione per avere immediatamente il desktop configurato come piace a noi, invece di ripristinarlo passo passo..

Per installare questa versione è necessario abilitare il repository testing di Ubuntu Tweak:

sudo add-apt-repository ppa:ubuntu-tweak-testing/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak

Attenzione perchè all'avvio il programma vi chiederà di abilitare il repository stabile del software, ovviamente non abilitatelo se non volete possibili conflitti di versione..
