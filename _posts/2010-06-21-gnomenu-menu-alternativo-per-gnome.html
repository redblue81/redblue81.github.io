GnoMenu è un menu alternativo per Gnome, altamente personalizzabile e con un set di temi ed icone forniti di default con cui ci si può sbizzarrire come meglio di crede. Si può ottenere un menu simile ad esempio a quello di KDE, oppure, perchè no, anche simile a quello di Windows 7..

Non solo, oltre ai temi forniti, se ne possono aggiungere di nuovi ed è possibile anche personalizzare i comandi da eseguire in corrispondenza di alcuni pulsanti, oppure si possono usare le icone di sistema in sostituzione di quelle di un determinato tema, in modo da integrarlo meglio nel nostro desktop..

L'installazione è semplicissima:

sudo add-apt-repository ppa:gnomenu-team/ppa && sudo apt-get update && sudo apt-get install gnomenu
