Installando una Lubuntu 11.10, mi è capitato per sbaglio di disabilitare il menù contestuale (quello che si apre col tasto destro del mouse per capirci) relativo al desktop. In pratica, di default la pressione del tasto destro sul desktop invoca il seguente menù:

Selezionando Preferenze del Desktop e poi la scheda Avanzate, avremo la possibilità di scegliere se visualizzare il menù di default (quello in figura), o quello di default di OpenBox, il DE di Lubuntu (attraverso il check dell'opzione Mostra il menù fornito dal window manager quando clicchi sul desktop). Quest'ultimo menù consente di arrivare direttamente alle impostazioni del DE, ma è (secondo me) meno "comodo" di quello di default (oltre che anche visivamente meno gradevole). Purtroppo di default non esiste la possibilità di tornare indietro, quindi sembrerebbe che, una volta scelto il secondo tipo di menù, sia necessario tenerlo.

Invece no! Basta dare il seguente comando:

pcmanfm --desktop-pref

Per avere nuovamente la possibilità si scegliere (andiamo sulla scheda Avanzate nella finestra che si aprirà e deselezioniamo l'opzione attivata in precedenza).  A questo punto, se il menù non torna subito a posto, basta riavviare.
