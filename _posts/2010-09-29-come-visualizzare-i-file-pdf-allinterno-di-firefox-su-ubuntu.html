Avete presente il plugin di Adobe Reader per Firefox su Windows? Quello che permette di "integrare" il lettore di file PDF nel browser.. Al contrario di default Ubuntu prevede la presentazione di una finestra in cui scegliere se aprire il documento nel visualizzatore predefinto (Evince), oppure se scaricarlo..

E' possibile però simulare il comportamento del plugin di Adobe, semplicemente installando MozPlugger.. Questo utilissimo plugin permette di visualizzare all'interno del browser numerosi file multimediali, ovviamente sfruttando programmi esterni.. Grazie al forum italiano di Ubuntu, vediamo come usarlo con i PDF..

Per prima cosa installiamo MozPlugger:

sudo apt-get install mozplugger

A questo punto dobbiamo creare il file di configurazione: nella directory relativa al nostro utente scriviamo

gedit .mozilla/mozpluggerrc

All'interno del documento appena creato scriviamo:

application/pdf:pdf:PDF file
application/x-pdf:pdf:PDF file
text/pdf:pdf:PDF file
text/x-pdf:pdf:PDF file
application/x-postscript:ps:PostScript file
application/postscript:ps:PostScript file
repeat swallow(evince) fill: evince "$file"
repeat swallow(documentShell) fill: acroread5 -geometry +9000$
repeat swallow(acroread) fill: acroread7 -openInNewWindow "$f$
repeat noisy swallow(kpdf) fill: kpdf "$file"
repeat noisy swallow(Xpdf) fill: xpdf -g +9000+9000 "$file"

Se non avevate altri plugin per i PDF allora avete finito, altrimenti cercate il file pluginreg.dat e cancellatelo. Ora riavviate Firefox e nella scheda Modifica -> Preferenze -> Applicazioni verificate che sia presente l'associazione con i file PDF.. Se esiste tale associazione, avete finito..

Alla prossima..
