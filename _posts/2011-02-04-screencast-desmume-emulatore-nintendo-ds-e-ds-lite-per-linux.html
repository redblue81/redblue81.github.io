DeSmuME è un emulatore di Nintendo DS e DS Lite multipiattaforma. Basterà installarlo sul vostro sistema ed avere la Rom del vostro gioco per poterlo usare sul vostro pc!!

E' possibile anche selezionare opzioni più avanzate per migliorare l'esperienza di gioco (in base alle vostre esigenze e all'hardware a vostra disposizione), ma per un utilizzo di base davvero è impossibile sbagliare, data la semplicità di utilizzo del programma.

EDIT: è presente anche nei repository di Ubuntu, quindi per installarlo date un:

sudo apt-get install desmume

Grazie a napulenhio per la segnalazione.. ;)

Qui il link di download. Per usarlo su Linux scaricate i sorgenti, poi semplicemente compilateli.. ;)

Nel mio caso come dipendenze è stato necessario installare i pacchetti libsdl1.2-dev e libglade2-dev, quindi può essere una buona idea dare, prima di compilare, il comando:

sudo apt-get install libsdl1.2-dev libglade2-dev

Poi decomprimete l'archivio scaricato, entrate nella directory desmume-0.9.7 ed eseguite i soliti:

./configure
make
sudo make install

Potete lanciare DeSmuME dando desmume da terminale, oppure potete crearvi un lanciatore apposito. Ho realizzato un piccolo screencast che dimostra l'emulatore in funzione. Mi spiace se c'è qualche rallentamento di troppo, ma avevo un tantino troppe applicazioni in running mentre lo realizzavo.. ;)
