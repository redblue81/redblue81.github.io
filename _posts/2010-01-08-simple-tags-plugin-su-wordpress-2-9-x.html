Recentemente ho aggiornato Wordpress alla versione 2.9.. Tutto ok, ma mi sono presto reso conto che il plugin Simple Tags aveva smesso di funzionare.. Situazione assolutamente risolvibile, poichè il problema è dovuto semplicemente al controllo di versione, che ancora non è stato aggiornato alla 2.9.. Allora per risolvere:

  • dalla dashboard di WP andate in Plugin -> Editor;
  • selezionate in alto a destra Simple Tags e cliccate su Seleziona;
  • cercate la riga

    if ( strpos($wp_version, '2.7') !== false || strpos($wp_version, '2.8') !== false ) {

  • e sostituitela con

    if ( strpos($wp_version, '2.7') !== false || strpos($wp_version, '2.8') !== false || strpos($wp_version, '2.9') !== false ) {

  • salvate cliccando su Aggiorna File.

Fatto.. ;)
