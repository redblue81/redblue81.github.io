Ieri è stata rilasciata, come di consueto, la nuova release di Ubuntu, la 11.10 Oneiric Ocelot; con la main release, sono state rese disponibili anche le immagini relative alle altre versioni, Lubuntu (con LXDE come Desktop Environment), Xubuntu (con XFCE) e Kubuntu (con KDE).

Questo il link per scaricare la main release: http://www.ubuntu.com/download

Questo invece il link per scaricare tutte le altre versioni: http://cdimage.ubuntu.com

Da dire che adesso è possibile installare Gnome-shell senza dover rinunciare a Unity, in quanto con il passaggio a Gnome 3 nativo, è tutto integrato nei repository. Quindi per installare Gnome-shell basta digitare:

sudo apt-get install gnome-shell

E successivamente riavviare la sessione scegliendo GNOME dal relativo menù. Consiglio inoltre l'installazione di Gnome-Tweak-Tool, anch'esso ormai presente nei repository:

sudo apt-get install gnome-tweak-tool

Infine, per i nostalgici, è ancora possibile installare la versione "classica" di Gnome, in questo modo:

sudo apt-get install gnome-session-fallback

Non dimenticate poi di riavviare la sessione e dal relativo menù scegliere GNOME Classic..
