Orta e Atolm.. Due ottimi temi per il vostro desktop, direttamente tramite PPA

Posted by RedBlue on January 15, 2011 · 1 min read

Orta e Atolm sono due temi, sviluppati dallo stesso autore, per il desktop di Ubuntu. Si tratta di due temi completi, veramente ben fatti e decisamente godibili alla vista. Inoltre, cosa che non fa mai male, sono davvero semplici da installare. Praticamente avviene tutto tramite PPA.

E' necessario installare, oltre ai temi in questione, anche il motore GTK di Equinox e, se volete, i set di icone di default consigliati dall'autore. Iniziamo dai repository:

sudo add-apt-repository ppa:tiheum/equinox (per il motore)
sudo add-apt-repository ppa:nikount/orta-desktop (per i temi)
sudo add-apt-repository ppa:alecive/antigone (per le icone)
sudo apt-get update

A questo punto, potete installare tutto il necessario:

sudo apt-get install gtk2-engines-equinox (motore GTK2)
sudo apt-get install atolm-theme orta-theme
sudo apt-get install faenza-icon-theme (set di icone consigliato per Orta)
sudo apt-get install awoken-icon-theme (set di icone consigliato per Atolm)

Volendo, potete installare anche i decoratori per Emerald e/o xfwm4:

sudo apt-get install orta-emerald-decorators
sudo apt-get install orta-xfwm4-decorators
sudo apt-get install atolm-emerald-decorators
sudo apt-get install atolm-xfwm4-decorators
