Primi passi con Ubuntu 11.04 Natty Narwhal - Parte 2

Posted by RedBlue on May 12, 2011 · 1 min read

Dopo l'installazione di ciò che ritengo fondamentale, oggi completo questa guida indicando alcuni altri software che uso ormai da tempo, e che ritengo affidabili ed accattivanti..

Innanzi tutto, direi di iniziare dai software di comunicazione:

sudo add-apt-repository ppa:hotot-team && sudo apt-get update && sudo apt-get install hotot
sudo apt-get install skype

Se usate MSN poi potete installare Emesene, giunto alla versione 2:

sudo apt-get install emesene

Ovviamente non possono mancare software come aMule, JDownloader, Gimp, indicator-weather, gdebi, più l'applet di notifica di GMail nella barra superiore:

sudo apt-get install amule gimp indicator-weather gdebi gm-notify
sudo add-apt-repository ppa:jd-team/jdownloader && sudo apt-get update && sudo apt-get install jdownloader

E, perchè no, anche Dropbox, per il quale basta scaricare il pacchetto .DEB adatto alla vostra distribuzione direttamente dal sito ufficiale.

Personalmente, sono un accanito sostenitore del browser di Google, Chrome, e per questo installo la Daily Build, dal seguente PPA:

sudo add-apt-repository ppa:chromium-daily/ppa
sudo apt-get update
sudo apt-get install chromium-browser chromium-browser-l10n

Infine, potete abilitare il repository Medibuntu, contenente molti pacchetti utili al corretto utilizzo di file multimediali:

sudo wget --output-document=/etc/apt/sources.list.d/medibuntu.list http://www.medibuntu.org/sources.list.d/$(lsb_release -cs).list && sudo apt-get --quiet update && sudo apt-get --yes --quiet --allow-unauthenticated install medibuntu-keyring && sudo apt-get --quiet update
sudo apt-get --yes install app-install-data-medibuntu apport-hooks-medibuntu
