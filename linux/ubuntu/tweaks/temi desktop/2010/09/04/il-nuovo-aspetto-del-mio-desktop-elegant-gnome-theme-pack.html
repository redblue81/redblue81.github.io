Il nuovo aspetto del mio desktop: Elegant Gnome Theme Pack

Posted by RedBlue on September 04, 2010 · 1 min read

Vi piace come si presenta in questi giorni il desktop del mio pc??

Si tratta del theme pack Elegant per Gnome, giunto alla versione 0.7 e oggi comprendente anche una personalizzazione per Firefox. Si tratta di un pack che modifica l'intero aspetto del vostro desktop, compresi i font e l'aspetto di alcune applicazioni..

Per installarlo da repository procedete in questo modo. In primo luogo avrete bisogno dell'ultima versione del Murrine Engine, che potete installare dal PPA apposito:

sudo add-apt-repository ppa:murrine-daily/ppa
sudo apt-get update
sudo apt-get install gtk2-engines-murrine

L'autore dell'Elengant Pack consiglia poi di installare Nautilus Elementary (ne ho già parlato qui), perciò:

sudo add-apt-repository ppa:am-monkeyd/nautilus-elementary-ppa
sudo apt-get update
sudo apt-get upgrade
nautilus -q #restarts nautilus

Adesso è tutto pronto per l'installazione dell'Elegant Gnome Theme Pack:

sudo add-apt-repository ppa:elegant-gnome/ppa
sudo apt-get update
sudo apt-get install elegant-gnome ttf-droid

NB. L'Elegant Theme Pack è disponibile anche per Linux Mint, l'installazione è semplicissima, sostituite l'ultimo dei tre precedenti comandi con questo:

sudo apt-get install elegant-gnome-mint ttf-droid

A questo punto, basta andare in Applicazioni -> Accessori -> Elegant GNOME e selezionate le opzioni che preferite..

Infine, per chi fosse interessato ad ulteriori personalizzazioni, basta andare sulla pagina del progetto su Gnome Look..

Alla prossima..
